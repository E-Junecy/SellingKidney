# 自制游戏：卖肾 v1.1 beta

回合制对战小游戏，玩家与 AI 进行 1v1 战斗，通过选择不同招式组合击败对手。

## 游戏规则

### 基础属性

| 属性 | 初始值 | 上限 | 说明 |
|------|--------|------|------|
| 血量 (HP) | 1 | 2 | 小于等于 0 时阵亡 |
| 气量 (Qi) | 0 | 无上限 | 使用招式消耗，一般不超过 256 |
| 护盾 (Shield) | 0 | 无上限 | 受到伤害时优先扣除 |

### 招式列表

| 招式 | 类型 | 气量消耗 | 效果 |
|------|------|----------|------|
| 聚气 | 增益 | 0 | 气量+1 |
| 治疗 | 增益 | 0 | 血量+0.5（不超过上限） |
| 叠盾 | 增益 | 1 | 护盾+1 |
| "酒" | 增益 | 1 | 下次攻击招式伤害+1，不可叠加 |
| 空气炮 | 攻击 | 0 | 对敌方造成 0.5 点伤害 |
| "嘣" | 攻击 | 1 | 对敌方造成 1 点伤害 |
| "劈" | 攻击 | 2 | 对敌方造成 1 点伤害 |
| "砍" | 攻击 | 3 | 对敌方造成 1.5 点伤害 |
| 防御 | 防御 | 0 | 抵消气量消耗 <3 的攻击招式 |
| 超级防 | 防御 | 1 | 抵消气量消耗 <6 的攻击招式 |
| 反弹 | 特殊 | 0 | 将敌方攻击伤害返还，使用后需冷却 1 回合 |
| 破反 | 特殊 | 0 | 若敌方使用反弹，则敌方永久失去反弹能力 |

### 特殊规则

1. **相同招式抵消**：双方使用相同招式时，无事发生
2. **不同攻击碰撞**：双方使用不同攻击招式时，按等效气量判定（实际气量消耗 + 酒增益×1.2），等效气量高的一方攻击成功；等效气量相同时双方抵消。酒增益不参与防御判定，即无法帮助攻击破防
3. **护盾吸收**：受到伤害时，护盾优先扣除与伤害点数相同的护盾值
4. **反弹冷却**：使用反弹后需冷却 1 回合，冷却期间无法使用反弹
5. **破反永久生效**：破反成功后，敌方永久失去反弹能力（本局内）
6. **防御上限**：防御只能挡住气量消耗 <3 的攻击，无法挡住"砍"及更强力的招式；超级防可挡住 <6 气量消耗的攻击
7. **酒的增益**：使用"酒"后获得"下次攻击招式伤害+1"增益，直到下次使用攻击招式结算完毕后消失；"酒"增益在攻击对攻时额外增加1.2等效气量（帮助赢得对攻），但不改变实际气量消耗，因此不能帮助破防；增益不可叠加，已有增益时无法再次使用"酒"

## AI 难度

| 难度 | 策略 |
|------|------|
| 简单 | 随机出招 |
| 普通 | 分析最近 3 次对战历史，根据玩家倾向调整策略，Softmax概率选取 |
| 困难 | 分析最近6次对战历史，根据玩家倾向加权评估，Softmax概率选取 |

## 游戏流程

```
主菜单
├── 开始游戏
│   ├── 选择难度（简单/普通/困难）
│   ├── 战斗（回合制）
│   │   ├── 显示双方状态（HP/Qi/Shield）
│   │   ├── 列出可用招式（气量不足的不显示）
│   │   ├── 玩家选择招式 → AI 根据策略出招
│   │   ├── 解算结果并输出解说
│   │   └── 一方阵亡时结束
│   ├── 保存对战记录到文件
│   └── 选择再来一局 / 回到主菜单
├── 查看历史记录
├── 清除历史记录
└── 退出
```

## 历史记录

- 每场对战结束后自动保存到 `history/` 目录
- 文件名格式：`时间_结果.txt`（如 `2025-07-10_19-30-00_win.txt`）
- 记录内容：每回合双方招式、结果解说、回合结束时的双方状态
- 主菜单提供一键清除所有历史记录功能

## 文件结构

```
SellingKidney/
├── main.py       # 入口，主菜单
├── game.py       # 战斗逻辑（回合结算、动画）
├── moves.py      # 招式定义
├── fighter.py    # Fighter 角色类
├── ai.py         # AI 策略（简单/普通/困难）
├── ui.py         # 战斗 UI、结果展示
├── console.py    # 控制台底层操作（光标、绘图、按键）
├── history.py    # 对战历史记录管理
└── history/      # 对战历史记录（自动创建）
```

## 下载

前往 [Releases](https://github.com/E-Junecy/SellingKidney/releases) 下载最新版 `SellingKidney.exe`，双击即可运行，无需安装 Python。

历史记录会自动保存在 exe 同级目录的 `history/` 文件夹中。

## 从源码运行

```bash
python main.py
```

无需安装额外依赖，仅使用 Python 标准库。
