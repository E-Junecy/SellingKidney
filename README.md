# 卖肾 - 回合制对战游戏

玩家与 AI 进行 1v1 战斗，通过选择不同招式组合击败对手。

## 游戏规则

### 基础属性

| 属性 | 初始值 | 上限 | 说明 |
|------|--------|------|------|
| 血量 (HP) | 1 | 2 | 小于等于 0 时阵亡 |
| 气量 (Qi) | 0 | 无上限 | 使用招式消耗 |
| 护盾 (Shield) | 0 | 无上限 | 受到伤害时优先扣除 |

### 招式列表

| 招式 | 类型 | 气量消耗 | 效果 |
|------|------|----------|------|
| 聚气 | 增益 | 0 | 气量+1 |
| 治疗 | 恢复 | 0 | 血量+0.5 |
| 空气炮 | 攻击 | 0 | 0.5 点伤害 |
| 嘣 | 攻击 | 1 | 1 点伤害 |
| 劈 | 攻击 | 2 | 1 点伤害 |
| 砍 | 攻击 | 3 | 1.5 点伤害 |
| 防御 | 防御 | 0 | 抵消 ≤3 气的攻击 |
| 超级防 | 防御 | 1 | 抵消 ≤6 气的攻击 |
| 反弹 | 特殊 | 0 | 反弹伤害，冷却 1 回合 |
| 破反 | 特殊 | 0 | 永久禁用敌方反弹 |
| 叠盾 | 增益 | 1 | 护盾+1 |
| "酒" | 增益 | 1 | 下次攻击伤害+1 |

### 特殊规则

1. **相同招式抵消**：双方使用相同招式时无事发生
2. **攻击碰撞**：不同攻击按等效气量判定，高者命中；相同则抵消
3. **护盾吸收**：伤害优先扣除护盾
4. **酒增益**：下次攻击伤害+1，攻击对攻时额外增加1.2等效气量

## AI 难度

| 难度 | 策略 |
|------|------|
| 简单 | 随机出招 |
| 普通 | 分析最近 3 次历史，根据玩家倾向加权选招 |
| 困难 | 分析最近 6 次历史，更准确判断玩家倾向 |

## 运行

```bash
python main.py
```

或自行打包为 exe：

```bash
# 1. 安装 PyInstaller
pip install pyinstaller

# 2. 在项目目录下执行打包（--onefile 生成单个 exe）
pyinstaller --onefile --console --name SellingKidney main.py

# 3. 打包完成后，在 dist/ 目录下找到 SellingKidney.exe
```

打包后将 `SellingKidney.exe` 复制到任意文件夹即可运行，历史记录会自动保存在 exe 同级目录的 `history/` 文件夹中。

## 文件结构

```
SellingKidney/
├── main.py        # 入口
├── game.py        # 对战逻辑
├── fighter.py     # 角色类
├── ai.py          # AI 策略
├── minimax.py     # minimax 算法
├── moves.py       # 招式数据
├── ui.py          # 界面显示
├── history.py     # 历史记录管理
└── history/       # 对战记录（自动创建）
```
